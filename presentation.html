Core Features

Everything you need, built-in

From patient booking to doctor management, AppointMedic covers the full lifecycle of a healthcare appointment system.

📅
Doctor Appointment Booking
Patients can browse doctors by speciality, view availability, and book appointments with a streamlined multi-step flow.
🤖
MediBot AI Chatbot
A draggable AI assistant powered by Google Gemini 2.0 Flash. Helps users navigate the platform and answers health-related queries in real-time.
🔐
JWT Authentication
Secure, role-based authentication for Patients, Doctors, and Admins using signed JSON Web Tokens and protected API routes.
📊
Admin Dashboard
Full administrative control — manage doctors, view all appointments, track earnings, and monitor platform-wide activity.
👨‍⚕️
Doctor Portal
Dedicated dashboard for doctors to manage their appointments, update profile/availability, and view patient details.
💳
Online Payments
Integrated payment gateways with Razorpay and Stripe for secure, flexible appointment fee collection online.
🖼️
Cloudinary Media Storage
Doctor profile images and patient uploads are handled via Cloudinary for fast, reliable cloud-based image management.
👤
Patient Profile Management
Patients can manage their personal profiles, view appointment history, and cancel bookings — all from one place.
🔍
Speciality-Based Search
Filter doctors by speciality (Dermatologist, Neurologist, Gynecologist, etc.) with intuitive navigation and doctor cards.

User Roles

Three portals, one platform

AppointMedic supports three distinct user roles, each with a dedicated interface and permissions.

👤 Patient User
  • Register & log in securely
  • Browse doctors by speciality
  • Book & cancel appointments
  • Pay online via Razorpay / Stripe
  • View & edit personal profile
  • Chat with MediBot AI assistant
👨‍⚕️ Doctor Provider
  • View upcoming appointments
  • Mark appointments complete / cancel
  • Update availability & fees
  • Manage doctor profile & photo
  • View dashboard earnings summary
  • Access patient booking details
🛡️ Admin Admin
  • Add & remove doctors
  • View all appointments system-wide
  • Access platform earnings
  • Manage doctor list & profiles
  • Dashboard with key statistics
  • Full administrative control
